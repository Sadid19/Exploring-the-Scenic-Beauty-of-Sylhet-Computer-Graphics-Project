# Exploring-the-Scenic-Beauty-of-Sylhet-Computer-Graphics-Project
This project takes you on a "Virtual journey" through some of the most beautiful and iconic landmarks of  Sylhet, Bangladesh. It showcases the natural and urban beauty of Sylhet by exploring landmarks such as: - Railway Station - Cha Bagan (Tea Gardens) - Jaflong - "Uttmachora" Waterfall  - Sylhet City
