# Exploring the Scenic Beauty of Sylhet: A Virtual Journey

## Description:
This project takes you on a **virtual journey** through some of the most beautiful and iconic landmarks of **Sylhet**, Bangladesh. It showcases the **natural** and **urban** beauty of Sylhet by exploring landmarks such as:
- **Railway Station**
- **Cha Bagan (Tea Gardens)**
- **Jaflong**
- **“Uttmachora” Waterfall**
- **Sylhet City**

The project is designed to give an immersive experience, exploring the cultural and natural diversity of Sylhet.

## Project Idea:
The **Exploring the Scenic Beauty of Sylhet** project aims to virtually display some of the most beautiful locations in the Sylhet region.

## Technologies Used:
- **OpenGL** for rendering graphical elements.
- **C++** as the programming language.
- **GLUT** for handling the window and user input events.
  
## Features:
- **Day and Night Mode**: Toggle between day and night using the **'N'** key.
- **Car Movement, Boat, train, cloud, birds**
  - **Keywords and Mouse used for controlling the objects**
